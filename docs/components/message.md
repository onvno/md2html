# message控件

用于即时信息的提示，消息背景色取决于消息类型，易可添加相对应的`icon`

# 如何使用

给父元素添加`.u-message`类

自定义类型背景  例如消息内容为`!new`的父元素添加`.u-mesnews`类

# 示例
