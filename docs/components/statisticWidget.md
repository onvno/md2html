# 数据统计卡片

数字统计卡片由一些[辅助类样式](http://design.yyuap.com/dist/pages/global-style/utilities.html)、[色彩](http://design.yyuap.com/dist/pages/global-style/color.html)、`u-counter`等样式实现。




## 多尺寸数据统计卡片

* `.u-counter-lg` - 大尺寸卡片
* `.u-counter-md` - 中尺寸卡片
* `.u-counter-sm` - 小尺寸卡片
